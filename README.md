# Altino Construções

Site Next.js 15 responsivo para empresa de bâtiment/rénovation em França.

## Instalar

```bash
npm install
npm run dev
```

Depois abre `http://localhost:3000`.

## O que está incluído

- Logo ALTINO em SVG
- CTA principal `DEVIS GRATUIT`
- Design mobile-first responsivo
- Menu mobile hamburger sem JavaScript extra
- Galeria com fotos remotas otimizadas pelo Next/Image
- Serviços de construção/rénovation
- Formulário visual de contacto
- Botão WhatsApp flutuante
- SEO básico no `app/layout.tsx`
